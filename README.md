## How to install

- downloading 
```
git clone  https://github.com/BGI-Qingdao/TGSGapFiller.git YOUR-INSTALL-DIR
```

- compiering

```
cd YOUR-INSTALL-DIR/main_src_ont
make 
```

## Usage 

- cp pipeline/pipeline.sh into your-working-directory
- edit pipeline.sh for your project
- run pipeline.sh
